Direct Download Methodology

Learn how to download specific dataset files efficiently using CLI tools or your browser, without cloning entire repositories.

GitHub Single-File Download

GitHub makes it easy to download individual files if you know where to look. To avoid cloning the whole repo, use Raw URLs.

Browser Method UI
  1. Find your file (e.g., data.csv) in the GitHub repository.
  2. Click the **Raw** button at the top right of the file preview.
  3. Right-click anywhere and select **Save As...** (or press **Ctrl+S**).
CLI Method Terminal
curl -L -o filename.csv "https://raw.githubusercontent.com/user/repo/branch/path/filename.csv"

Use -L to follow redirects and -o to specify the output name.

HuggingFace Direct Download

HuggingFace datasets can be downloaded directly using Resolve URLs which bypass the data explorer UI.

Resolve URL Format HF
https://huggingface.co/datasets/{user}/{repo}/resolve/main/{filename}

You can use this URL format directly with wget or curl.

Wget Example Terminal
wget https://huggingface.co/datasets/glue/resolve/main/README.md

Kaggle CLI Integration

For Kaggle, the most reliable way to download specific files without the whole archive is the Official Kaggle CLI.

Download Single File CLI
kaggle datasets download -d {user}/{repo} -f {filename}

Note: This requires a kaggle.json API key in your user directory.

Platform Setup (Tool Installation)

If you don't have curl or wget installed, use these commands to set them up quickly using your platform's package manager.

Windows (Chocolatey) Tool
choco install curl wget
Windows (Scoop) Tool
scoop install curl wget
macOS (Homebrew) Tool
brew install curl wget

Enable "Save As" Popup

If you prefer to choose exactly where each file is saved on your system, you can enable this feature in your browser settings once for all downloads.

Google Chrome & Microsoft Edge
  • Go to Settings > Downloads.
  • Enable "Ask where to save each file before downloading".
Mozilla Firefox
  • Go to Settings > General > Files and Applications.
  • Select "Always ask you where to save files".
Direct Saving: Modern browsers like Chrome and Edge now support the FileSystem Access API, which DataNest uses to trigger a system "Save As" dialog automatically for most datasets!
